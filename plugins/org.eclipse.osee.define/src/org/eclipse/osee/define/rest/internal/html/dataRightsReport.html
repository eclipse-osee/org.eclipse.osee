Ryan D. Brooks Data Rights Report Generates the Data Rights Report
  • Selected Branch - provide the Branch to obtain the Data Rights Report for
  • Code Root Directory - if specified, provides:
    • a file listing the directories to search
    • a directory to recurse into looking for files
  • Start Button - generates the report


function oseeAppStart(oseeAppParams) { var branch = oseeAppParams["selected_branch"]; var rootDir = oseeAppParams["code_root_dir"]; window.location.assign("../datarights?branch="+branch+"&code_root="+rootDir); }
