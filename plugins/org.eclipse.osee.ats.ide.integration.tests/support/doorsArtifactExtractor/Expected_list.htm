3.1.2.4.1.2 Voice status.
The following voice warning status shall be provided for each crewmember:
  1. Aircraft voice warning. An audio indication to alert the crew of detected EXA threats, including the following:
    1. Countermeasures maneuver cue (if enabled).
    2. Threat type.
    3. Threat azimuth.
    4. Threat mode.
  2. EXA voice threshold. An indication as to the current voice threshold mode selected.
  3. Maneuver cue. An indication as to whether the EXA maneuver cue voice warning is on or off.
  4. HFI voice selection. An indication as to whether HFI voice warning is on or off.
